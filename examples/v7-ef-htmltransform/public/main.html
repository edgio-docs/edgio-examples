Examples using HtmlTransformer

This page contains a list of examples that demonstrate how to use the HtmlTransformer within edge functions. Each example fetches a static HTML asset and modifies the content and response headers.

  • Basic Example

    This example demonstrates a basic usage of HtmlTransformer within edge functions. This code fetches a static HTML asset and modifies the content and response headers.

    curl -v http://localhost:3000/basic_example
  • ESI Example

    This example shows how to use Edge Side Includes (ESI) with edge functions. Similar to the basic example, this code fetches a static HTML asset and looks for ESI tags to include additional content. These includes are fetched and processed by the edge function before responding to the client.

    curl -v http://localhost:3000/esi_example
  • ESI Response Stream Example

    This example illustrates how to use ESI with a streamed response to the HtmlTransformer. It's the same as the ESI example, but the response of the initial fetch request is streamed to the transformer.

    curl -v http://localhost:3000/esi_response_stream_example
